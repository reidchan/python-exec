> 执行python脚本并返回结果。

**具体使用请查阅[文档](https://www.yuque.com/super2god/open-source/python-exec-v0.0.2)**

# 示例代码
```js
import { execByText } from 'python-exec';

(async () => {
  const result = await execByText(`print('hello guy');`);
  console.log('result =>', result);
})();

// 运行结果：result => hello guy...
```
